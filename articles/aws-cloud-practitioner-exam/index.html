How to Pass the AWS Certified Cloud Practitioner Exam

Steve Savstrom · published 22 AUGUST 2022

As a new software engineer I wanted to develop my cloud computing skills so that I had the ability to build and deploy modern web applications that were scalable, highly-available, and performant. As new engineer at a large company, I saw that there were many legacy applications that needed to be migrated to or rebuilt in the cloud.

I took my first step into cloud computing by taking the AWS Cloud Practitioner exam. The exam covers a wide range of topics, including different AWS services, security, architecture, pricing, and support. I want to share my experience and how I prepared for the exam.

The AWS Certified Cloud Practitioner exam assesses foundational skills in Amazon Web Services (AWS) with an emphasis on:

  • Benefits of cloud computing
  • AWS shared responsibility model
  • AWS Cloud costs, economics, and billing practices
  • Security best practices,
  • Core AWS services including compute, network, databases, and storage
  • Identifying AWS services for common use cases

The AWS Certified Cloud Practitioner exam covers the following domains:

  • Cloud Concepts (26% of exam):This domain covers basic cloud computing concepts, including the characteristics of cloud computing, the benefits of using the cloud, and the differences between different deployment models (e.g., public, private, and hybrid cloud).
  • Security and Compliance (25% of exam): This domain covers the security features and best practices for the AWS Cloud, including the AWS shared responsibility model, identity and access management, and data encryption.
  • Technology (33% of exam): This domain covers the key AWS services and features, including compute, storage, database, and network services. It also covers the AWS global infrastructure, including regions and availability zones.
  • Billing and Pricing (16% of exam): This domain covers the pricing models for AWS services and the tools and features available for managing and optimizing costs. Support: This domain covers the AWS support plans and the resources available for troubleshooting and getting help with AWS.

I studied for 3 weeks and spent about 20 hours on video courses, documentation, tutorials, and practice exams. I found these resources the most useful:

  • Pluralsight AWS Certified Cloud Practitioner (CLF-C01) Learning Path
  • AWS Cloud Practitioner Essentials course
  • Udemy AWS Certified Cloud Practitioner Practice Exams HQ Questions
  • Udemy AWS Certified Cloud Practitioner Practice Exams

The AWS Certified Cloud Practitioner exam is a multiple-choice and multiple-response exam, which means that it consists of questions that have one or more correct answers. The exam is delivered through a network of testing centers around the world, and it is administered using a computer-based testing (CBT) system. The exam consists of 65 questions, and individuals have 90 minutes to complete it. To pass the exam, individuals must score at least 720 out of 1000 points.

Overall, I'm happy that I chose to get certified! I'm excited to apply the skills in my current role and I'm motivated to keep learning and earning more certifications in the future.
